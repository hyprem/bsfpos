Module-scoped executor state (declared after the reducer section):

```js
// --- Executor state (module-scoped) ---
let currentState = STATES.BOOTING;
let deps = null; // { webContents, store, safeStorage, mainWindow, log }
const timers = Object.create(null);
let hasCreds = false;

const credentialsStore = require('./credentialsStore');
const adminPin = require('./adminPin');
```

**start(opts):**

1. Validate opts has webContents, store, safeStorage, mainWindow, log. Throw if missing.
2. Assign to module-scoped deps.
3. Compute initial hasCreds:
   - If !credentialsStore.isStoreAvailable(safeStorage) → fire `notify({ type: 'safestorage-unavailable' })` and return.
   - const loaded = credentialsStore.loadCredentials(store, safeStorage);
   - If loaded === credentialsStore.DECRYPT_FAILED → fire `notify({ type: 'decrypt-failed' })` and return.
   - hasCreds = loaded !== null;
   - Fire `notify({ type: 'creds-loaded' })` (reducer reads hasCreds via ctx).
4. Start the boot watchdog via the start-timer side-effect naturally emitted from the reducer — do NOT double-arm it manually.

**notify(event):**

1. Build ctx = { hasCreds }.
2. Call reduce(currentState, event, ctx) → { next, sideEffects }.
3. If next !== currentState: log `auth.state: ${currentState} -> ${next} reason=${event.type || 'unknown'}` via deps.log.info (the reducer ALSO emits log side-effects with reason, but this single-line transition trace is the AUTH-04 audit artifact).
4. Set currentState = next.
5. For each sideEffect in order: call _runSideEffect(sideEffect). Catch and log any thrown error per effect — never let one failing handler block the rest.

**_runSideEffect(effect) — dispatch table implementing the 03-04-SUMMARY contract:**

| kind | handler |
|------|---------|
| `log` | `deps.log.info('auth.state: ' + currentState + ' reason=' + effect.reason)` |
| `start-timer` | `timers[effect.name] = setTimeout(() => { delete timers[effect.name]; notify({ type: 'timer-expired', timer: effect.name }); }, effect.ms)` |
| `clear-timer` | `if (timers[effect.name]) { clearTimeout(timers[effect.name]); delete timers[effect.name]; }` |
| `fill-and-submit` | `const creds = credentialsStore.loadCredentials(deps.store, deps.safeStorage); if (!creds || creds === credentialsStore.DECRYPT_FAILED) { notify({ type: 'decrypt-failed' }); return; } const js = 'window.__bskiosk_fillAndSubmitLogin(' + JSON.stringify(creds.user) + ',' + JSON.stringify(creds.pass) + ')'; deps.webContents.executeJavaScript(js).catch((e) => deps.log.warn('fill-and-submit failed: ' + e.message))` |
| `show-credentials-overlay` | `deps.mainWindow.webContents.send('show-credentials-overlay', { firstRun: !!effect.firstRun })` |
| `hide-credentials-overlay` | `deps.mainWindow.webContents.send('hide-credentials-overlay')` |
| `show-pin-modal` | `deps.mainWindow.webContents.send('show-pin-modal')` |
| `hide-pin-modal` | `deps.mainWindow.webContents.send('hide-pin-modal')` |
| `show-error` | `deps.mainWindow.webContents.send('show-magicline-error', { variant: effect.variant })` |
| `clear-credentials` | `credentialsStore.clearCredentials(deps.store); hasCreds = false;` |
| `rerun-boot` | `const loaded = credentialsStore.loadCredentials(deps.store, deps.safeStorage); hasCreds = (loaded && loaded !== credentialsStore.DECRYPT_FAILED); notify({ type: 'creds-loaded' });` |

Unknown effect kinds: `deps.log.warn('auth.unknown-side-effect kind=' + effect.kind)` and continue.

**handleCredentialsSubmit({ user, pass, pin }):**

```js
async function handleCredentialsSubmit({ user, pass, pin }) {
  try {
    const record = adminPin.buildRecord(pin);              // pure
    const ciphertext = credentialsStore.buildCiphertext(   // pure
      deps.safeStorage,
      { user, pass }
    );
    // ATOMIC single set — D-11 requirement
    deps.store.set({
      adminPin: record,
      credentialsCiphertext: ciphertext
    });
    notify({ type: 'credentials-submitted' });
    return { ok: true };
  } catch (e) {
    deps.log.error('handleCredentialsSubmit failed: ' + e.message);
    if (e && e.code === 'safestorage-unavailable') {
      notify({ type: 'safestorage-unavailable' });
    }
    return { ok: false, error: e.message };
  }
}
```

**handlePinAttempt(pin):**

```js
function handlePinAttempt(pin) {
  try {
    if (adminPin.verifyPin(deps.store, pin)) {
      notify({ type: 'pin-ok' });
      return { ok: true };
    }
    notify({ type: 'pin-bad' });
    return { ok: false };
  } catch (e) {
    deps.log.error('handlePinAttempt failed: ' + e.message);
    return { ok: false, error: e.message };
  }
}
```
