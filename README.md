# Bee Strong POS Kiosk

Self-service POS kiosk for Bee Strong Fitness.
See `.planning/PROJECT.md` and `CLAUDE.md` for project details.

## Development

```bash
npm install
npm start       # dev mode (windowed, DevTools open)
npm run build   # produces Windows NSIS installer in dist/
```

Operator runbook: `docs/runbook/README.md`.
